## ✨ Overview

Welcome to **Cozy Cup Co.** — a luxurious, Awwwards-inspired landing page built to showcase the ultimate rich hot chocolate experience. 

This project explores advanced frontend mechanics like **Scrollytelling** using an **HTML5 Canvas sequence** synced perfectly to the user's scroll via **Framer Motion**. Text animations, smooth floating layouts, and pure deep black backgrounds meticulously merge to construct a hyper-realistic, luxury culinary aesthetic.

**Why is this project special?**
This entire sophisticated web application was fully developed and engineered using **AI assistant pairing**! It marks a monumental first step in understanding and orchestrating AI to produce high-end, production-ready frontend codebases seamlessly.

## 🎯 Features

- **Scroll-Linked Image Sequence:** Renders a 120-frame WebP sequence dynamically perfectly anchored to the viewport height.
- **Scroll Interpolation:** Silky smooth scrolling (no stutter) powered by Framer's `useSpring`.
- **Intelligent Loading States:** Prevents frame stuttering by completely pre-loading all sequence images into memory, masked behind an elegant custom loader.
- **Dynamic Transforms:** Elegant float-in/out parallax transformations orchestrating the storytelling "beats" seamlessly.
- **Responsive "Contain" Logic:** Dynamic HTML5 Canvas rendering scales flawlessly across Desktop, Tablet, and Mobile limits.

## 💻 Getting Started (Local Development)

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Soha-Hansa/cozy-cup
   ```

2. **Navigate into the directory:**
   ```bash
   cd cozy-cup
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Open [http://localhost:3000](http://localhost:3000) to view the application!
